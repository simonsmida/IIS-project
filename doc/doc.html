Implementace

Stručná dokumentace k implementaci, která popisuje, které části projektu (např. PHP skripty) implementují jednotlivé případy použití.

Projekt je založený na architektonickom vzore MVC, ktorý použitý framework Django implementuje naturálne. Pojekt je rozložený do viacerých aplikácií, ktoré implementujú dané prípady užití. Aplikácie su rozdelené do súborov models.py (Model), views.py (Controller) - tu sa deje väčšina logiky projektu, forms.py - vypĺňacie formuláre, url.py - routing, templates/*.html (View).

V hlavnej aplikácii "shelter" sú pomocou tried implementované modely, ktoré využívame v ostatných aplikáciach a všetka funkcionalita prihlasovania, registrácie úpravy účtu. Aplikácie medzi sebou komunikujú na základe URL reroutov.
Najväčšia koncentrácia funkcionality je v aplikácii "caregiver", ktorá implementuje tzv. Caregiver portal(pečovatelský portál) - tu sú implementované všetky prípady užitia pečovatela, tj. overovanie dobrovoľníkov, evidencia zvierat, vytváranie rozvrhov pre zvieratá, schvalovanie rezervácií, evidencia zapožičania/vrátenia zvieraťa a písanie požiadavkov na veterinára
Aplikácia myadmin implementuje základný pohľad adminovského portálu, avšak funkcionalita je implementovaná v aplikáciách - volunteer_edit, vet_edit, caregiver_edit.
Aplikáce "animals", "reservation" a "vetrequest" sú samovysvetľujúce. Deje sa tú všetka manipulácie z danými modelmi - CRUD operácie. Aplikácia "timetable" je určená pre pečovatela, ktorý ju používa na vytváranie rozvrhov pre zvieratá. Pri prihlásení ako dobrovoľník alebo veterinár sa na hornej lište zobrazí navigácia, ktorá implementuje ich prípady užití - rezervácie a história venčení (dobrovoľník); čakajúce požiadavky, vyšetrenia a vedenie zdravotných záznamov zvierat (veterinár). Neregistrovaný/neprihlásený užívateľ si môže prezerať web a informácie o zvieratách.

Instalace

Postup instalace na server:
  • Server: AWS Lightsail
  • DB: Amazon RDS
  • pripojenie RDS databázy na Lightsail inštanciu: Návod
  • inštalácia požiadavkov: nginx, gunicorn, Django v. 4.1.3
  • konfigurácia nginx: Videonávod
  • pip install -r requirements.txt
  • konfigurácia static súborov: Dokumentácia
  • gunicorn mysite.wsgi --bind 0.0.0.0:8000 --workers 4
Softwarové požadavky:
  • Django 4.1.3, Python >3.8, pip
  • asgiref==3.5.2
  • crispy-bootstrap5==0.7
  • django-crispy-forms==1.14.0
  • django-session-timeout==0.1.0
  • mysqlclient==2.1.1
  • six==1.16.0
  • sqlparse==0.4.3
  • tzdata==2022.6
Konfigurácia databázy:
  • python manage.py makemigrations shelter
  • python manage.py migrate
  • Poznámka: premazanie už vytvorených migrácií (pravdepobne), príkazy vyššie potom pravdepobne premažú DB
